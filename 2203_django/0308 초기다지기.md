## Django 처음 복습

### 환경 설정

1. 가상환경 실행

```bash
$ python -m venv venv
```

2. venv 폴더 안에있는 Scripts 폴더안에있는 activate를 실행시킨다.

```bash
$ source venv/Scripts/activate
(venv)
```

3. 만약에 git을 쓸거라면 : gitignore 폴더를 생성 => git을 쓰기 위해서 

```text
venv/
__pycache__/
```

4. 장고설치

```bash
$ pip install django==3.2.12
```

5. 프로젝트 및 앱 생성

- 프로젝트 만들 때 . 꼭 써라

```bash
$ django-admin startproject pjt .
```
